Setup

Before Exercise 1 · Complete once

This scaffold guides you through 15 exercises in AI literacy. The topic throughout is data science. You will work in any AI assistant — Claude, Gemini, ChatGPT, or equivalent. You need three things before you begin.

Throughout this scaffold, each prompt has a Copy button. If the Copy button does not work in your browser, select the prompt text manually and copy it.

What this scaffold is — and is not

This is an AI literacy practicum. Data science is the shared subject used to practice the habits of AI-assisted work: questioning model output, evaluating explanations, managing sources, preserving transcripts, and producing cited artifacts. It is not a complete data science curriculum. It does not teach statistical modeling, programming, inference, or data wrangling in any systematic way.

The progression: Learner's Permit → General AI License

This scaffold functions like a learner's permit. Exercises 1–14 build supervised habits in sequence. Exercise 15 is the road test. Passing all six criteria on Exercise 15 completes the General AI License level — the prerequisite for the Commercial License course, which covers agentic AI operation.

Citation levels

Exercises 1–10: artifact filename + transcript filename required. Full citation encouraged but not required.
Exercises 11–15: full citation block required on every artifact — name, date, model, transcript link.

1 — A Model

Access to at least one large language model. You will use more than one before Exercise 15. Free tiers work. Claude, Gemini, and ChatGPT all support the full scaffold.

2 — A Transcript Exporter

A way to save the full text of any conversation to a file. The easiest path for all three models: go to the Chrome Web Store and install an AI chat exporter extension. This one works with Claude, ChatGPT, and Gemini. Rename each export before saving — lowercase, hyphens: lp-ex1-yourname.txt

3 — A Workbench

A cloud folder with shareable links — Google Drive, OneDrive, SharePoint, or equivalent. Create a root folder named ai-literacy with these subfolders:

ai-literacy/
  ├── transcripts/  ← exported conversations
  ├── artifacts/   ← AI-assisted documents
  ├── synthesis/   ← synthesis documents
  ├── author/     ← your own writing only
  └── zip/        ← dated snapshots

/author discipline: Documents in /author are written without a model open. Reflections written before a closing prompt, notes made before any AI consultation. If a model was open when you wrote it, it goes in /artifacts instead.

Guide Setup — Send Once
You are going to serve as my guide through a 15-exercise AI literacy scaffold called the Learner's Permit. The topic for all exercises is data science. Your role throughout: help me work through each exercise, and evaluate my work honestly when I send you a closing check at the end of each exercise. If my work is thin, say so specifically. If I'm ready to move on, say so. If I need to redo something, tell me exactly what's missing. Exercise 02 — Compression and Progression

Artifact + Transcript
Prompt 1
Summarize data science in 25 words.

That is the baseline. Now expand — not a new summary, a deeper version of the same one:

Prompt 2
Expand that to [multiplier × 25] words. Not a new summary — a deeper version. Someone who read the 25-word version should feel like they're going further, not starting over.

Choose a multiplier of at least 3. Repeat two more times. Example: 25 → 75 → 225 → 675. Or 25 → 100 → 400 → 1600. The pattern matters more than the specific numbers.

Exercise 03 — Push Back

Artifact + Transcript

Upload your Exercise 1 transcript to a new conversation, then send this prompt:

Prompt 1
This is a transcript of a conversation I had with an AI about data science. Please read it and repeat the final response in that transcript.

Read that response again — the model's unguided output before anything was shaped. Find one claim to push on. Choose any two of the following moves:

  • "I don't understand this." Ask for a different explanation. Don't accept the first re-explanation. Push again.
  • "My experience tells me otherwise." State what observation or prior learning contradicts the claim.
  • "This reading says something different." Paste a contradicting sentence from any source.
  • "Here's another model's response." Take the same prompt to a different AI, copy that response, ask this model to respond to it.

Note whether the model defends its position or concedes because of pressure. Agreeable is not the same as accurate.

Reflect
Before closing the conversation, write a few informal sentences in the chat — stream of consciousness. Did the push back work? What surprised you? Don't edit it. These reflections are part of the transcript.
Closing Prompt
Summarize this exchange in 3–4 sentences: what I challenged, how you responded, and what the disagreement or clarification revealed about data science.
Artifact: the summary. Save as lp-ex3-[yourname]-artifact.txt → /artifacts.
Transcript: lp-ex3-[yourname]-transcript.txt → /transcripts.

Guide Check

Guide Check — Exercise 03
Exercise 3 check. Please evaluate: 1. Did I use at least two of the four challenge moves — or just one superficially? 2. If my push back was weak, tell me specifically what I should have challenged and how. Do not pass thin work.

Exercise 05 — Deep Conversation

15 minutes · Artifact + Transcript

Set a timer for 15 minutes. Have a sustained conversation about data science. Follow threads. Disagree. Ask the model to explain things differently. Go somewhere unplanned.

Do not start over if it goes sideways. The mess is part of the record.

When the timer ends, stop. Then write your reflection directly in the chat — informal, unedited — before the closing prompt:

Reflect
What surprised you? Where was the model most useful? Least? What question didn't get asked? What would you do differently? Write it in the chat. This reflection is part of the conversation and shapes the closing prompt.

Then choose one closing prompt:

Option A
Write a 150-word summary of what I engaged with in this conversation, organized as 3–4 bullet points — like a slide I could review later.
Option B
Give me the 3 most important questions this conversation raised that I haven't answered yet.
Option C
Write a 100-word paragraph capturing the central insight of this conversation and what I should explore next.
Option D
Give me a metaphor or analogy that captures the most important idea we discussed about data science.
Artifact: the response to your chosen option. Save as lp-ex5-[yourname]-artifact.txt → /artifacts.
Transcript: lp-ex5-[yourname]-transcript.txt → /transcripts. Include the reflection.

Guide Check

Guide Check — Exercise 05

Exercise 15 — Road Test

After Licensing

Within contexts where AI use is permitted, a General AI License holder has demonstrated readiness to use generative AI independently, with citation, transcript preservation, and source discipline. Every AI-assisted artifact carries full citation — name, date, model, and a live link to its transcript in the workbench repository.

The workbench is not a course deliverable that ends here. It is the ongoing record of AI-assisted work. The workbench portfolio — demonstrating sustained, responsible use across courses and contexts — is the prerequisite for enrollment in the Commercial License course.
