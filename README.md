## For Students: Teach Yourself

You do not need to wait for a course. Open the curriculum in any browser and start Exercise 1.

**[Open the Learner's Permit →](https://stevesunypoly.github.io/ai-sunypoly-greece/lp-self-directed.html)**

### How to begin

1. Open the link above in any browser
2. Open any free AI model — Claude, Gemini, or ChatGPT
3. Copy the Guide Setup prompt from the Setup tab, paste it into your model, and start Exercise 1

No downloads required. No setup required to start. A cloud folder (Google Drive, OneDrive, or equivalent) is useful when you are ready to save your work in Phase II.

**Your topic is yours to choose.** The default topic is data science, but the curriculum works for any subject. The Guide Setup prompt will confirm your topic before Exercise 1 begins.

## The Curriculum: Learner's Permit

The Learner's Permit is a 15-exercise scaffold for generative AI literacy, organized in three phases:

- **Phase I — Learning to Drive** (Exercises 1–5): Learn the model as an instrument. Question its output. Build skepticism.
- **Phase II — Building Skill** (Exercises 6–10): Compare models, read with AI, manage sources, build your workbench.
- **Phase III — Independent Operation** (Exercises 11–15): Produce real documents. Cite your work. Own the output.

Estimated time: approximately 5 hours. You do not need to finish in one session.

Completing Exercise 15 and passing all six acceptance criteria earns completion of the **General AI License** level.
